◆ getPrecedence()

int getPrecedence ( char  oprtr)

Returns an integer representing the precedence of the input operator.

Precedence used:

  1. ~
  2. *
  3. +
  4. >

-1 is returned if the oprtr is not one of the above

Parameters
[in]oprtrThe operator whose precedence is to be checked
Returns
An int representing the oprtr 's precedence

◆ getPrefix()

char * getPrefix ( char *  infixExp)

Returns the prefix form of the given infix expression.

Parameters
[in]infixExpThe proposition logic formula expression in infix form ( char array pointer)
Returns
The proposition logic formula expression in prefix form ( char array pointer)

Some notes about the input and the function:

  1. The input must be a well formed propositional logic formula expression in infix form
  2. All whitespace between characters will be ignored
  3. The formula need not be well bracketed, in case of missing brackets, precedence is calculated by getPrecedence()
  4. The expression is considered to be case sensitive (a and A are different atoms)

The memory for the output is dynamically allocated. stack data structure is used to store the operators untl they are needed
