int makeTreeFromPrefix ( struct node *  root,
char *  prefixExp 
)

Makes a rooted binary parse tree from the given prefix expression.

Parameters
[out]rootThe pointer to the root of the binary parse tree
[in]prefixExpThe proposition logic formula expression in prefix form ( char array pointer)
Returns
The length of the expression string used to make the tree (will be the length of the input prefix expression if it is well formed)

Some notes about the input and the function:

  1. The input must be a well formed propositional logic formula expression in prefix form
  2. All whitespace between characters will be ignored
  3. The expression is considered to be case sensitive (a and A are different atoms)
Note
The memory for the tree is dynamically allocated
